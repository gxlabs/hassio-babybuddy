# gxlabs Home Assistant add-ons

## Installation

1. Open Home Assistant → **Settings → Add-ons → Add-on Store**
2. ⋮ menu → **Repositories** → paste `https://github.com/gxlabs/hassio-babybuddy` → **Add**
3. The **Baby Buddy (gxlabs)** add-on will appear under this repository. Install and start it.

## Add-ons

- **[babybuddy](./babybuddy/)** — Baby Buddy with per-event image uploads.
